# Goal is to create a project with 4-5 mini games inside it for children age(4-10)

it should be as addictive as possible
colorful ,simple,not much brain teaser

TechStack
Frontend-> React native
Backend-> Node.js(simple rest apis (no realtime needed or online support))

Business Aspect-> we can later use ads inside the app for continue(revival) of player

---

- No login or authentication
- just simple install ,choose game and start playing
- Inside each game we have to keep track of highest score of that game until the point
- Should have sounds and haptics

## Game-1

- Screen will contain tiles of different colors-> the player objective will be to match those tiles without overlapping flow with another color

## Game-2

- Player will the archer which will shoot balloons comings towards it along with joker
- if hit the joker then player's movement will be slowed down
- if keeps on hitting the balloons points will be increased and speed also
- if any joker reaches the player then game over

## Game-3

- A player with a gun shooting a balloons (just like in our mela)
- the Player have only 3 chances if he miss a hit, but if he didn't than he can have a score card for highest,
- On initial the balloons are stationary but after scoring it get moving to make it hard

## Game-4

- Player with dart to hit the balloons
